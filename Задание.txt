1. Реализовать электронную очередь, в очередь добавляются игры (класс QueueGame);

2. Играть могут только зарегистрированные игроки.

3. У каждого игрока есть:
идентификатор, 
логин, 
количество сыгранных игр, 
список сыгранных игр и результаты.

4. Игрок может:
играть в игры из очереди, 
просмотреть список игр,
добавить игру в очередь,
зарегистрироваться, авторизоваться.

Результат игры изменяет состояние игрока.

4. Вывести рейтинг игроков:
по количеству сыгранных игр;
по количеству побед.

5. Список игроков хранится в файле.
